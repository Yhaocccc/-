##  遊戲介紹
「十點半」是一款撲克牌遊戲，遊戲目標是要讓手中的牌點數加總達到10.5點，並且比莊家（或其他玩家）的牌點數更接近10.5點，以獲得勝利。每張牌的點數計算方式為：A 為1點、2-10 以牌面點數計算、J、Q、K 則為0.5點。遊戲開始時，玩家可以先下注，接著莊家發牌，玩家可以選擇要牌或停牌，當所有玩家都停牌後，莊家會依照一定規則抽牌，最後比較點數決定輸贏。
